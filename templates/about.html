About

What We Do

Cinemite was created so that people could have a reference list of all the movies they love. You can find movies by browsing through our list of categories as well as search for films if you already know what you want. You can share your opinions of those movies and read other people's reviews by going to the movie's details page.

Find out about our team below!

Technologies Used

TMDB

This is the API we used to gather info on all your favorite movies.

Python

Used to construct our 'back end' processes.

Javascript

Used to make our front end API calls and add dynamic interfaces.

Bootstrap

Used to add custom classes and styling to our page.

SQLite

Used to build our database of users, comments, and movie lists.

Github

Used as a collaboration environment where we can edit and share code.
